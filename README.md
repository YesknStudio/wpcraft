# wpcraft
wordpress博客托管和博客交流分享社区

> 这个仓库不是wpcraft的源码仓库，wpcraft项目目前不是开源的，如果你需要和wpcraft类似的社区程序，请参考这个仓库： [Vmoex](https://github.com/yeskn-studio/vmoex)  
> 本仓库仅做为wpcraft的版本更新迭代、缺陷管理、计划发布等功能  
> 谢谢你关注wpcraft，祝你的博客更加精彩！
